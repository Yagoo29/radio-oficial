# 📻 Rádio Oficial FM 87.9 - Iguape/SP

Site oficial da **Rádio Comunitária FM 87.9**, localizada em Iguape - São Paulo.  
O projeto foi desenvolvido com HTML, CSS e JavaScript puro, com foco em simplicidade, leveza e acesso rápido ao streaming da rádio.

---

## 🚀 Funcionalidades

- 🎧 Player de áudio integrado (streaming ao vivo)
- 🔴 Indicador de "AO VIVO" automático
- 📱 Layout responsivo (funciona em celular e PC)
- 📻 Informações da rádio e programação
- 📸 Link para Instagram oficial
- 📰 Seção de informações e notícias da rádio
- 🌙 Design moderno em tema escuro

---

## 🖥️ Tecnologias usadas

- HTML5
- CSS3 (Flexbox + Grid + Animações)
- JavaScript (controle de status ao vivo)
- Google Fonts (Poppins)
